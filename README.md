## Installation

```
npm install turtler --save
```

## Usage

> the given options are the defaults

```javascript
let table = new Turtler([
  ["uid", "name"],
  ["1", "Doe"],
  ["2", "Hemma"]
], {
  hasHeader: true,
  columnSeperator: ' | ',
  headerSeperator: '='
});

console.log(table);
```

This will yield:

```
uid | name
===========
1   | Doe  
2   | Hemma
```

### Markdown

> We can also output markdown tables just as easily


```javascript
let table = new Turtler([
  ["uid", "name"],
  ["1", "Doe"],
  ["2", "Hemma"]
]);

console.log(table.markdown());
```

This will yield:

```
| uid | name  |
|-|-|
| 1   | Doe   |
| 2   | Hemma |
```

And looks like when rendered:

| uid | name  |
|-|-|
| 1   | Doe   |
| 2   | Hemma |
